Post Widget to display recent posts.

== Description ==

Post Widget is a very powerful widget plugin for WordPress which displays your most recent or latest posts with thumbnails. This is perfect for simple blogs & online magazines. Widgets come with highly-customizable control panels. Easy to use and support multi-widget even in the same sidebar.






== Installation ==

1. Download the plugin.
2. Upload it to the plugins folder of your blog.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Now, goto the Widgets page of the Appearance section and configure the Post Widget.
